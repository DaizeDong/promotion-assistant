# scripts/apply.py — fork it (do not hand-write)

`promotion-assistant apply` delegates to THIS file to bridge `secrets/<slug>.env` into the active
`~/.claude.json` (it must never echo secret values). It is intentionally NOT generated by
init_config.py so there is a single source of truth: **fork it from `companion config kit`'s
`scripts/apply.py`** and adapt the per-channel KEY=VALUE mapping. See ../runbooks/new-machine.md.
